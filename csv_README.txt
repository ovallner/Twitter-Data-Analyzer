CSV Readme

COLUMN: DATA
A   :   Tweet text
B   :   Author of tweet's screen name
C   :   Time tweet was created
D   :   hashtags contained in tweet
E   :   urls contained in tweet
F   :   mentions in tweet
G   :   id number of tweet
H   :   (if it exists) id of tweet this tweet is replying to
I   :   (if it exists) screen_name of person this tweet is replying to
J   :   (if it exists) country of author of this tweet
K   :   (if it exists) indicates this tweet is a retweet. contains screen name of author of original tweet
